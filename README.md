
### 3. 개발자 모드 활성화
오른쪽 상단의 **개발자 모드**를 켭니다.

### 4. 압축해제된 확장 프로그램 로드
**압축해제된 확장 프로그램을 로드합니다** 버튼을 클릭한 뒤, 프로젝트 폴더를 선택합니다.

---

## 사용 방법

1. 치지직 라이브 페이지를 엽니다.
2. 브라우저 툴바에서 확장 프로그램 아이콘을 클릭합니다.
3. 채널 ID가 자동으로 감지되면 그대로 사용하고, 감지되지 않으면 직접 입력합니다.
4. 아래 항목을 설정합니다.
    - **갱신 주기**: 몇 초마다 `liveId`를 확인할지
    - **새로고침 대기**: 변경 감지 후 몇 초 뒤 새로고침할지
5. **시작** 버튼을 누릅니다.
6. 팝업에서 현재 모니터링 상태, `liveId`, 마지막 갱신 시각을 확인할 수 있습니다.
7. 중단하려면 **중지** 버튼을 누릅니다.

---

## 예시 사용 시나리오

- 방송 페이지를 열어둔 상태에서 스트림 세션이 갱신됨
- 기존 페이지가 더 이상 최신 방송 상태를 반영하지 않음
- 확장 프로그램이 `liveId` 변경을 감지
- 몇 초 후 자동 새로고침 수행
- 사용자는 최신 방송 화면으로 자동 복귀

## 주의 사항

- 치지직 API 응답 형식이 변경되면 정상 동작하지 않을 수 있습니다.
- 너무 짧은 체크 주기를 사용할 경우 불필요한 요청이 많아질 수 있습니다.
- 확장 프로그램은 치지직 서비스 구조에 의존하므로, 사이트 변경 시 수정이 필요할 수 있습니다.
